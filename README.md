# muvluvgg-translation
Bản dịch máy cho game MuvLuv girls garden (DMM Player). 
# Hướng dẫn cài đặt :

1: vào mục release, tải muvluv melon về, giải nén ra thư mục game (nơi chứa file muv_luv_girlsgardenx_cl.exe hoặc muv_luv_girlsgarden_cl.exe)

2: Khởi động game như bình thường từ DMM Player hoặc icon trên desktop

3: Enjoy
