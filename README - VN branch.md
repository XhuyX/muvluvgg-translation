# MuvluvGG Translation (MuvluvMod)

Bản dịch máy và mod hỗ trợ cho game MuvLuv Girls Garden.
*Các file trong kho lưu trữ này dành riêng cho client game phiên bản DMM Game Player trên hệ điều hành Windows.*

## Tính năng nổi bật

* **Cung cấp bản dịch cốt truyện** (Bao gồm cốt truyện chính và cốt truyện sự kiện nhân vật).
* **Loại bỏ Mosaic** (che mờ) được thêm vào tự động trong game.
* **Luôn bật nút Bỏ qua (Skip)**.
* **Không ngắt quãng giọng nói** (Voice) khi chuyển thoại trong cốt truyện.
* **Tự động bỏ qua (Auto-skip) chiến đấu**.

## Hướng dẫn sử dụng

### 1. Chuẩn bị
* Đảm bảo bạn đã cài đặt client game (Phiên bản DMM Game Player).
* Tìm thư mục chứa file thực thi của game (thường là `muv_luv_girlsgardenx_cl.exe` hoặc `muv_luv_girlsgarden_cl.exe`).

### 2. Tải Plugin
* Truy cập vào trang **[Releases](../../releases)** của kho lưu trữ này.
* Tải xuống file nén của phiên bản mới nhất (ví dụ: `MuvluvMod-Melon.zip`).

### 3. Cài đặt
* **Giải nén file vừa tải về rồi vứt toàn bộ nội dung vào thư mục game là được.**
* *(Lưu ý: Các file/thư mục giải nén ra phải nằm cùng chỗ với file `muv_luv_girlsgardenx_cl.exe` của game. Nếu hệ thống hỏi có muốn ghi đè file cũ không, hãy chọn Đồng ý/Replace).*

### 4. Khởi động game
* **Khởi động game như bình thường**: Hãy mở game thông qua DMM Game Player hoặc shortcut trên Desktop. (KHÔNG nhấp đúp trực tiếp vào file `muv_luv_girlsgardenx_cl.exe` để mở!).
* Trong lần khởi động đầu tiên, một cửa sổ console (bảng điều khiển đen) có thể sẽ hiện lên để tải một số thư viện cần thiết. Hãy kiên nhẫn chờ đợi, sau khi hoàn tất game sẽ tự động mở.

## Phím tắt trong game

* **F2**: Bật / Tắt dịch thuật
* **F3**: Bật / Tắt luôn hiển thị nút Skip
* **F4**: Bật / Tắt ngắt quãng giọng nói (Voice)
* **F5**: Bật / Tắt tự động bỏ qua chiến đấu

## Tuyên bố miễn trừ trách nhiệm

* Plugin này là sản phẩm của bên thứ ba (fan-made) và **không có bất kỳ mối liên hệ nào** với nhà phát triển hay nhà phát hành chính thức của game.
* Plugin này chỉ được cung cấp cho mục đích học tập và nghiên cứu công nghệ. Vui lòng sử dụng nó một cách hợp pháp và tuân thủ các quy định.
* Việc sử dụng plugin này có thể ảnh hưởng đến hoạt động bình thường của game. Tác giả **không chịu trách nhiệm** cho bất kỳ vấn đề nào phát sinh từ việc sử dụng plugin này (bao gồm nhưng không giới hạn ở việc bị khóa tài khoản, mất dữ liệu, hoặc lỗi phần mềm).
* Khi tải xuống và sử dụng plugin này, đồng nghĩa với việc bạn đã đồng ý tự chịu mọi rủi ro liên quan.
